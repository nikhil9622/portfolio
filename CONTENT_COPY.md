### **Project 1: ZERA - Agentic Security Chatbot**

**Project Title:**
```
ZERA - Zero-Endpoint Risk Agent
Agentic Security Chatbot for Endpoint Risk Remediation
```

**Overview:**
```
ZERA is an intelligent security agent that monitors endpoint health, 
scores risks in real-time, maps threats to MITRE ATT&CK tactics, and 
executes remediation plans with user approval. Built with LLM reasoning, 
it bridges the gap between threat detection and automated response.
```

**The Problem:**
```
Traditional endpoint security tools generate alerts but require manual 
investigation and remediation. Security teams are overwhelmed with 
alerts, and response times lag‚Äîallowing threats to persist.
```

**The Solution:**
```
ZERA automates the entire threat response lifecycle:
1. **Real-Time Risk Scoring** - Analyzes endpoint signals continuously
2. **MITRE ATT&CK Mapping** - Maps threats to known tactics (e.g., T1036, T1105)
3. **Severity Banding** - Categorizes risks (Low, Medium, High, Critical)
4. **AI-Driven Remediation** - LLM generates structured JSON remediation plans
5. **User Approval Workflow** - Human-in-the-loop for critical actions
6. **Automated Execution** - Endpoint agent runs allow-listed fixes
7. **Immutable Audit Trail** - Logs every action (prompt ‚Üí plan ‚Üí result)
```

**Key Features:**
- ‚úÖ Real-time endpoint risk scoring
- ‚úÖ MITRE ATT&CK framework integration
- ‚úÖ LLM-powered remediation planning
- ‚úÖ Human-in-the-loop approval system
- ‚úÖ Automated fix execution (DNS flush, process kill, Defender updates)
- ‚úÖ JWT-based org-scoped access control
- ‚úÖ Complete audit trail for compliance

**Tech Stack:**
```
Backend: Python
AI/LLM: GPT-based reasoning engine
Security: MITRE ATT&CK API, JWT authentication
Endpoint: Windows Defender, DNS, Process Management
Database: MongoDB (audit logs)
```

**Impact:**
```
üìâ Reduced response time from hours to minutes
üéØ 90% of low/medium threats auto-remediated
üîí 100% audit compliance with immutable logs
‚ö° Real-time threat visibility across endpoints

ML-Powered Phishing Email Detection
Protecting Users from Social Engineering Attacks

```
A machine learning system that detects phishing emails by analyzing 
textual content and metadata. Uses SVM and Random Forest classifiers
